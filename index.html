Interested in Learning More About Linux?

Hey, you! I'm starting a GNU/Linux User Group at George Mason University this semester, Fall 2013! We need you to help cultivate and grow Mason's free software community.

MasonLUG will provide a vehicle to learn about Linux and free software in a social setting, create and contribute to free software, and advocate for the use of Linux and free software on campus.

If you're a Linux newbie, an open-source geek, a student looking for projects to hack on, a free software philosopher, or someone who's just plain interested in learning what the hubub around Linux and free software is about, we need you!
